# robosys2023
ロボットシステム学のプログラム

## プログラムの使い方
plus
標準入力から読み込んだ数字を足すプログラム。
小数,整数に対応しています。
bashに
seq 10 | ./plus
と入力すると55.0と返ってきます。

## 必要なソフトウェア
* Python
  * テスト済み: 3.7〜3.10
